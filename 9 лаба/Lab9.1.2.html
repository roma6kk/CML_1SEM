HTML (HyperText Markup Language) - это стандартный язык разметки для создания содержания веб-страниц. Версия HTML5 предоставляет множество новых элементов и атрибутов, которые улучшают структуру и функциональность веб-страницы. Это включает в себя: Семантические элементы (например, header, nav, article, section, footer), которые помогают браузерам и поисковым системам понимать структуру страницы. Мультимедийные элементы (например, video и audio), позволяющие вставлять видео и аудио без сторонних плагинов. Графические элементы (например, canvas и svg), которые позволяют создавать интерактивные графику. API для хранилища данных, такие как Local Storage и Session Storage, для сохранения данных на стороне клиента.
CSS (Cascading Style Sheets) - это язык для стилизации веб-страниц. CSS3 представляет набор новых функций и возможностей, включая: Гибкие блочные модели с использованием Flexbox и Grid, которые позволяют легко управлять расположением элементов на странице. Анимации и переходы, которые позволяют создавать плавные и интерактивные эффекты на веб-страницах. Градиенты и тени для создания более креативных и привлекательных дизайнов. Медиазапросы, которые позволяют создавать адаптивные дизайны, которые корректируются в зависимости от размера экрана устройства. Поддержка шрифтов - CSS3 предоставляет возможность встраивать шрифты на веб-страницу с помощью @font-face, что позволяет создавать уникальные и креативные дизайны с разнообразными шрифтами. Вместе HTML5 и CSS3 позволяют создавать современные веб-сайты, которые выглядят красиво, работают хорошо на различных устройствах и предоставляют богатый пользовательский опыт
